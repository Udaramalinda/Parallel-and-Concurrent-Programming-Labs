Step 1: Compilation
To compile the code, you will need to run the make command.

For Windows users: Comment out the Linux make command in the Makefile and uncomment the Windows-specific make command.
For Linux users: Comment out the Windows make command in the Makefile and uncomment the Linux-specific make command.
Once you have made the appropriate changes, you can run the make command in the terminal. The compiled executables will be located in the build folder.

Step 2: Visualization
Next, open the vis.ipynb file in Jupyter Notebook. Run each cell in the notebook sequentially to generate the graphs displaying the results.
